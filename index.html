Calculation details

The tool computes cumulative ascent by adding only positive elevation changes between consecutive GPX points. For example, +800 m, -100 m, +200 m counts as 1000 m of ascent.

Distance is computed from GPS coordinates using the haversine formula. Average grade is cumulative ascent divided by segment distance. Max grade is the steepest positive grade over a short rolling window along the VK segment, which reduces one-point GPS/elevation noise.
